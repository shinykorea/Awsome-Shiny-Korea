# 어썸 샤이니 코리아 앱스

끝내주게 멋있는(어썸한) 한국 R [Shiny](https://shiny.rstudio.com/) 사용자들이 제작한 애플리케이션 목록과 한국어 자료를 제공하는 프로젝트입니다. 본 프로젝트에 기여하시고 싶으면 풀리퀘스트나 [이슈](https://github.com/shinykorea/Awsome-Shiny-Korea/issues)를 등록해 주세요.
